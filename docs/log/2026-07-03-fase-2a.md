# Log Kerja — Fase 2a: PWA Penuh & Cetak Faktur

**Tanggal:** 3 Juli 2026 · **Status akhir:** selesai, merged ke `main`

## Yang dibangun

- **PWA penuh** (vite-plugin-pwa, mode generateSW/Workbox, autoUpdate):
  - Aplikasi kini **bisa di-install** di Android/Windows/macOS/iOS (ikon PNG 192/512 + maskable dibuat dari logo SVG).
  - **Offline-capable**: app shell (12 entri, ±488 KiB) tercache oleh service worker — aplikasi tetap terbuka tanpa internet; permintaan `/api` sengaja selalu ke jaringan agar data keuangan tidak pernah basi.
  - Update otomatis: versi baru terpasang sendiri saat pengguna membuka aplikasi.
- **Cetak / PDF faktur**: halaman cetak khusus tanpa shell aplikasi (`/cetak/faktur?tenant=&id=`) — kop perusahaan (nama, alamat, NPWP dari pengaturan), nomor faktur, pelanggan, rincian barang, PPN, total, status LUNAS/BELUM LUNAS; tombol "Cetak / Simpan PDF" memakai dialog print browser. Tombol 🖨 Cetak di setiap faktur penjualan.

## Validasi (semua hijau)

- Typecheck · 16 unit test · build (sw.js + manifest + precache dihasilkan).
- **Smoke test 75 pemeriksaan** (2 baru): manifest.webmanifest & sw.js terlayani Worker (200); seluruh 73 pemeriksaan lama tetap lulus.
- Playwright: halaman cetak faktur dirender benar (kop PT Maju Jaya + NPWP, INV-00001, PPN 11%, total Rp 499.500); `navigator.serviceWorker` terkonfirmasi **registered** di browser sungguhan.
